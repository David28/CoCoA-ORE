This folder contains tools and notes that I wrote to help me analyze this dataset.

Description of the files contained here:

The file DifferentLabelReasons.txt explains the reasons why certain examples were labelled differently.

 - The following files contain one file name per line:
 - DIFFERENT_LABELS.txt: contains the names of all files that we labelled different from SARD;
 - GET_DIRECT.txt: Contains the names of all code examples that get input from $_GET directly;
 - GET_VIA_ARRAY.txt: Contains the names of all code examples that get input from $_GET via an array;
 - POST_DIRECT.txt: Contains the names of all code examples that get input from $_POST directly;
 - POST_UNSERIALIZE.txt: Contains the names of all code examples that get input from $_POST and unserialize it;
